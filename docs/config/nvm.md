# NVM（Node Version Manager）配置

NVM是一个用于管理Node.js版本的工具。它允许您在同一台计算机上同时安装多个Node.js版本，并且能够轻松切换这些版本以适应不同的项目需求。NVM在开发中尤其有用，因为不同的项目可能需要不同版本的Node.js来运行。通过NVM，您可以在项目之间自由切换Node.js版本，而无需重新安装或卸载。
