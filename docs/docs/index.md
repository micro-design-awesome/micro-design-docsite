# MicroDocs Framework
## 项目简介
[MicroDocs Framework](https://microdesign.gitee.io/micro-design-docs/) 是一套基于MicroDesign体系的文档平台，致力于从用户体验、需求分析、系统架构、系统插件、系统组件、系统主题、系统测试、系统部署、开发文档和解决方面，解决企业痛点，为企业降本增效的的生态解决方案。
