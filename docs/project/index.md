## 前端技术栈

### 核心框架
- **Vue 3**: 渐进式 JavaScript 框架
- **React**: 用于构建用户界面的 JavaScript 库
- **Angular**: 企业级前端框架

### 构建工具
- **Vite**: 下一代前端构建工具
- **Webpack**: 模块打包工具
- **Rollup**: JavaScript 模块打包器
- **Parcel**: 零配置构建工具

### 状态管理
- **Pinia**: Vue 3 的状态管理库
- **Vuex**: Vue 官方状态管理模式
- **Redux**: 可预测的状态容器
- **MobX**: 简单、可扩展的状态管理

### 样式解决方案
- **Tailwind CSS**: 实用优先的 CSS 框架
- **UnoCSS**: 即时原子化 CSS 引擎
- **Sass/Less**: CSS 预处理器
- **CSS Modules**: CSS 模块化方案

### 开发工具
- **TypeScript**: JavaScript 的超集
- **ESLint**: JavaScript 代码检查工具
- **Prettier**: 代码格式化工具
- **Husky**: Git hooks 工具
